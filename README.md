# 기록물 정리사업 공정관리 대시보드

## 실행 방법

### 1. 필수 패키지 설치
```bash
pip install streamlit pandas plotly openpyxl
```

### 2. (선택) 기존 xlsx 데이터 가져오기
```bash
cd d:\noah\dashboard
python load_existing_data.py
```
> 통합공정진행표(20251126).xlsx의 데이터를 대시보드에 로드합니다.

### 3. 대시보드 실행
```bash
streamlit run d:\noah\dashboard\app.py
```
> 브라우저에서 http://localhost:8501 로 접속됩니다.

## 기능

| 메뉴 | 기능 |
|------|------|
| 📈 대시보드 | 전체 공정 현황, 공정별 게이지, 일별 추이 차트 |
| ✏️ 일일 실적 입력 | 작업자별 일일 공정 실적 입력/관리 |
| 👥 작업자별 현황 | 작업자별 실적, 일평균 생산성 분석 |
| 🔍 샘플링 검사 | 품질검사 입력, 오류율/품질률 추이, 적합/부적합 판정 |
| ⚙️ 설정 | 사업정보, 목표량, 작업자 관리, 데이터 백업/복원 |

## 데이터 파일
- `data.json` : 모든 데이터가 저장되는 파일 (자동 생성)
- 설정 > 데이터 관리에서 JSON 백업/복원 가능
